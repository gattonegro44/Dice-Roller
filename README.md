# 🎲 Dice Roller

A simple random dice simulator implemented in multiple programming languages.

---

## 📖 Overview

This repository demonstrates random number generation using:

- JavaScript
- Python
- C#
- PHP
- Go

Useful for:

- Learning randomness
- Utility applications
- Game development basics
- Beginner projects

---

## ✨ Features

- Simulates a six-sided dice
- Random number generation
- Console-based output
- Multi-language examples

## Running

### JavaScript

```bash
node dice.js
```

### Python

```bash
python dice.py
```

### C#

```bash
dotnet run
```

### PHP

```bash
php dice.php
```

### Go

```bash
go run main.go
```
